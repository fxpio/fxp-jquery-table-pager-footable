Getting Started
===============

Prerequisites
-------------

This library require:

- jquery
- footable
- sonatra-jquery-table-pager
- bootstrap (optional)
- font-awesome (optional)

Installation
------------

Installation is a quick:

Add library in your `bower.json`:

```json
{
    "dependencies": {
        "sonatra-jquery-table-pager-footable": "latest"
    }
}
```

Launch bower:

```bash
$ bower install
```
