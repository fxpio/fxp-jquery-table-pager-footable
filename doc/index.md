Getting Started
===============

Prerequisites
-------------

This library require:

- jquery
- footable
- sonatra-jquery-table-pager
- bootstrap (optional)
- font-awesome (optional)

Installation
------------

```
npm install @sonatra/jquery-table-pager-footable --save
```

Dev installation
----------------

### Use NPM

```
npm install
```

### Use grunt

```
grunt serve
```

For ajax example, you must run this command in parallel:

```
grunt nodemon
```
